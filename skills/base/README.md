# Base

面向基础环境、服务器连接、设备管理与容器运维的开发入口目录。

当前 `skills/base/` 已承载真实 skill 目录；请按下面链接进入对应 skill 开发：

- [`npu-smi/`](npu-smi/)：设备查询、健康状态、温度/功耗、固件与虚拟化
- [`ascend-docker/`](ascend-docker/)：容器环境与设备映射
- [`torch_npu/`](torch_npu/)：PyTorch 昇腾扩展、环境检查与迁移指引
- [`remote-server-guide/`](remote-server-guide/)：远程服务器 SSH、容器连接、文件传输与故障排查
- [`npu-docker-launcher/`](npu-docker-launcher/)：昇腾 NPU 服务器 Docker 容器一键启动
- [`ascend-dmi/`](ascend-dmi/)：NPU 硬件状态、带宽、算力、功耗、诊断与复位
- [`ascend-avi-vnpu/`](ascend-avi-vnpu/)：AVI 模式与 vNPU 实例管理

推荐场景：

- 新同学基础环境搭建
- 服务器连接、容器开发环境准备
- 设备可用性、硬件性能与虚拟化状态检查

对应 bundle：`ascend-base`
