# Huawei Ascend Ecosystem Components
# 华为昇腾生态组件

## 1. CANN (Compute Architecture for Neural Networks)

- **说明**：昇腾计算架构
- **功能**：提供 AI 算子库和编译工具链
- **版本**：CANN 7.0+ 推荐

## 2. MindSpore

- **说明**：华为自研 AI 框架
- **功能**：与 msmodelslim 深度集成
- **支持**：模型训练和推理

## 3. ModelArts

- **说明**：华为云 AI 开发平台
- **功能**：支持一站式模型开发、部署

## 4. Ascend CL

- **说明**：昇腾计算语言
- **功能**：提供 C/C++/Python 接口
