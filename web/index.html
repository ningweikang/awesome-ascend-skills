如何下载 Skill

页面不会直接替你安装文件，而是帮你生成可复制的安装命令或 Agent 安装说明。

1

搜索并查看 Skill

在 Skill 索引里输入关键词，点击左侧条目后，右侧会展示 SKILL.md 的详细说明。

2

选择要安装的 Skill

勾选左侧复选框，或在右侧详情点击“选择此 Skill”。可以单选、多选，也可以全选当前列表。

3

复制安装命令

到下方“批量安装命令”复制 npx 命令，或复制给 Agent 的安装说明。
